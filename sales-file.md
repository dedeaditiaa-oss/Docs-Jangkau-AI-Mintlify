# 📂 File Basis Pengetahuan Sales

Fitur **File** pada Bot Sales (*Chat Commerce*) memungkinkan Anda untuk memberi bot anda dengan dokumen referensi produk, profil bisnis, serta kebijakan operasional bisnistoko Anda. Dengan mengunggah dokumen, Bot AI akan memiliki pengetahuan dasar yang kuat untuk mengenali produk, meyakinkan calon pembeli, serta melayani pertanyaan transaksional secara mandiri.

![Halaman Pengaturan File Sales Bot](/assets/sales-file-1.png)

---

## 📄 Jenis Dokumen Sales yang Direkomendasikan

Untuk membantu Bot AI meningkatkan angka konversi penjualan dan melayani pembeli dengan maksimal, Anda disarankan mengunggah file tertulis seperti:

* **Profil Perusahaan** Informasi latar belakang bisnis, keunggulan produk, sejarah singkat perusahaan, atau nilai jual unik (*Unique Selling Point*) dari toko Anda.
* **Brosur & Deskripsi Produk Detail:** Penjelasan mendalam mengenai bahan baku, varian warna/rasa, fungsi, cara perawatan, serta instruksi khusus penggunaan produk.
* **Syarat & Ketentuan Pembelian:** Aturan transaksi di toko Anda, jam operasional pengiriman, batas waktu transfer pemesanan, atau kebijakan ganti rugi jika barang rusak di perjalanan.
* **Kebijakan Pengembalian & Garansi:** Informasi mendetail mengenai apakah toko Anda menerima penukaran barang (*retur*), refund dana, atau klaim garansi produk tertentu.

---

## ⚙️ Ketentuan & Format File

Agar file basis pengetahuan ini dapat dibaca dan dipelajari dengan oleh bot AI, pastikan file Anda memenuhi standar berikut:

* **Ukuran Maksimal:** Ukuran file yang diunggah tidak boleh lebih dari **20 MB** per dokumen.
* **Format yang Didukung:** Format file dalam bentuk **`.pdf`** dan **`.docx`**.

---

## 📤 Cara Mengunggah Dokumen

1. Masuk ke menu **Bot AI** pada panel kiri dasbor -> klik tombol **Edit** pada Bot Sales Anda.
2. Klik menu **File** yang terletak di panel konfigurasi sebelah kiri.
3. Pada area kotak di tengah, Anda bisa langsung **menyeret dan melepas (*drag & drop*)** file dari komputer, atau **klik area tersebut** untuk memilih file secara manual dari folder penyimpanan laptop Anda.
4. Setelah file terpilih, detail indikator data berupa total *Karakter* dan file *Pending* yang terekstraksi akan terbaca di kartu ringkasan sebelah kanan.
5. Klik tombol hijau **Simpan** untuk mengunggah dokumen tersebut ke dalam ingatan Bot AI.

---

⚠️ **Catatan Penting:** Pastikan seluruh teks di dalam file `.pdf` atau `.docx` Anda merupakan teks ketikan asli (bukan teks dalam gambar hasil foto atau hasil *scan* ). Dokumen dengan susunan tata bahasa yang rapi, bersih dari coretan gambar latar belakang, dan menggunakan poin-poin terstruktur akan sangat membantu AI memahami spesifikasi produk Anda dengan tingkat akurasi yang jauh lebih presisi.
